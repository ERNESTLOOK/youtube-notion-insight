# 글로벌 AI 트렌드 채널 가이드

요청하신 취지("글로벌 AI 트렌드를 가장 빨리 따라잡기")에 맞춰, 해외·국내 유튜브와 팟캐스트 중 **속도·신뢰도·활용도**가 높은 채널만 골라 정리했습니다.  
아래 채널들을 "데일리 브리핑 → 주간 큐레이션 → 심층 인터뷰 → 한국어 보완" 레이어로 조합해서 들으시면, 교수·작가·협회장으로서 쓰실 자료와 인사이트를 매우 빠르게 축적하실 수 있을 것입니다.

> **이 문서 활용법**  
> - 각 채널 설명 아래 `📡 채널 ID`, `👥 규모`, `🔄 업로드 빈도` 뱃지를 달았습니다.  
> - 마지막 섹션 **"Notion 등록 통합표"** 를 그대로 복사해 `channel_discover.py`의 채널 목록으로 사용할 수 있습니다.

## AI 슬롭(slop) 회피를 위한 채널 선택 팁

- 한국은 AI로 양산된 저품질 영상(소위 'AI 슬롭') 소비·생산 비중이 세계적으로 매우 높은 국가라는 조사 결과가 반복해서 보고되고 있습니다.
- 유튜브·틱톡 등은 AI 슬롭 채널에 대해 채널 삭제·수익 정지 등 강경 대응을 확대하고 있어, **채널 신뢰도와 제작자 정체성**을 기준으로 선별하는 것이 중요합니다.
- **선별 기준:** ① 제작자 실명·소속 명확 ② 출처 표기 있음 ③ 정기적 업로드 이력 6개월 이상 ④ 커뮤니티 반응(댓글)에 실질 논의 있음

### 채널 ID 확인 방법

`(확인 필요)` 항목은 아래 두 가지 방법으로 빠르게 확인할 수 있습니다.

**방법 1 — YouTube에서 직접 확인:**
1. YouTube에서 핸들(@xxx)로 채널 접속
2. URL에서 `/channel/UCxxxxxxx` 형식이면 그것이 채널 ID
3. `/@handle` 형식이면 채널 소개 페이지 소스에서 `channelId` 검색

**방법 2 — 프로젝트 내 자동 조회:**
```python
# youtube_client.py의 resolve_channel_id() 활용
from src.youtube_client import YouTubeClient
yt = YouTubeClient(api_key="YOUR_KEY")
print(yt.resolve_channel_id("@matthew_berman"))  # → UCawZsQWqfGSbCI5yjkdVkTA
```
